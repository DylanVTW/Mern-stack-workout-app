Installatie Instructies

1. open de terminal (... linksboven of Crtl+Shift+`)
2. run deze command (npm install)
3. maak een .env file aan met daarin een PORT, MONGO_URI en NODE_ENV
4. maak een gitignore aan met daarin het volgende: node_modules/, .env, .DS_store, Thumbs.db, .vscode/, .idea/


Hoe de app te starten
1. open de terminal en toets in cd backend
2. toets daarna in npm run dev
3. maak een tweede terminal aan en toets in cd frontend
4. toets in npm run dev
5. click op de port van de localhost
